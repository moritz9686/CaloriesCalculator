# Calories Tracker

Small full-stack calorie tracker with:

- React web frontend
- Express API
- MongoDB persistence
- Email/password authentication

## Setup

1. Copy `.env.example` to `.env`
2. Install dependencies with `npm install`
3. Start the API with `npm run server`
4. Start the frontend with `npm run client`

## Features

- Sign up and log in
- Save daily calories consumed and burned
- Use manual totals and itemized entries together
- View net calories for a selected date
